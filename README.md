## Dev Setup

### Docs

1. Go to the docs directory : `cd docs`
2. Run `pnpm i --ignore-workspace`
3. Run `pnpm run docs:dev`

### Package

1. Clone the repo
2. Run `pnpm i`
3. Run `pnpm story:dev` to run histoire (storybook)
4. Open `http://localhost:6006`
